CSS 动画

CSS 动画（CSS Animation） 是现代网页设计中不可或缺的一部分，它能够提升用户体验、增强视觉效果，让页面更加生动有趣。CSS 提供了多种方式来实现动画效果，包括 transition 和 animation 属性。

CSS 动画的两大类别

1. Transition（过渡）

transition 用于在元素状态改变时添加平滑的过渡效果，比如鼠标悬停、焦点变化等。

核心属性

  • transition-property：指定要过渡的 CSS 属性
  • transition-duration：过渡持续时间
  • transition-timing-function：过渡的时间函数（缓动效果）
  • transition-delay：延迟开始时间
.box {
  width: 200px;
  height: 200px;
  background-color: orange;
  transition: width 2s ease 0s;
}

.box:hover {
  width: 400px;
}

与 Transform 相关的属性

  • transform：用于元素的2D或3D转换（如平移、旋转、缩放、倾斜）
  • transform-origin：定义变换的基点（原点），默认为元素中心点（50% 50%）

transform-origin 属性允许你修改变换的中心点，默认情况下，变换发生在元素的中心点（50% 50%）。你可以使用关键词（如 top, right, bottom, left, center）或具体的坐标值（如 px, %）来指定变换的原点。

.element {
  transform-origin: top left; /* 左上角为变换中心 */
  transform: rotate(45deg); /* 将围绕左上角旋转 */
}

支持过渡的属性

  • 尺寸：width, height, margin, padding
  • 颜色：background-color, color, border-color
  • 形状：border-radius, transform
  • 透明度：opacity

2. Animation（动画）

animation 用于创建更复杂的动画序列，不需要用户交互即可自动播放。

核心属性

  • animation-name：指定关键帧名称
  • animation-duration：动画持续时间
  • animation-timing-function：动画时间函数
  • animation-delay：延迟开始时间
  • animation-iteration-count：播放次数（infinite 表示无限循环）
  • animation-direction：播放方向（normal, reverse, alternate, alternate-reverse）
  • animation-fill-mode：动画完成后的状态（forwards, backwards, both）
  • animation-play-state：播放状态（running, paused）

Transition 实际应用示例

以下是一些常见的 transition 应用场景：

  • 尺寸变化

    .box1 {
  width: 200px;
  height: 200px;
  background-color: orange;
  transition: width 5s linear 0s;
}

.box1:hover {
  width: 800px;
}

  • 位置移动

    .box2 p {
  position: relative;
  left: 0;
  transition: left 1s linear 0s;
}

.box2:hover p {
  left: 1000px;
}

  • 颜色变化

    .box3 {
  background-color: red;
  transition: background-color 1s linear 0s;
}

.box3:hover {
  background-color: green;
}

  • 形状变换

    .box4 {
  border-radius: 0;
  transition: border-radius 1s linear 0s;
}

.box4:hover {
  border-radius: 50%;
}

  • Transform 组合效果

    .box5 {
  transition: transform 1s linear 0s;
}

.box5:hover {
  transform: scale(1.2) rotate(360deg);
}

  • Transform-origin 应用示例

    .box9 {
  transition: transform 1s linear 0s;
  transform-origin: bottom right; /* 设置变换中心点为右下角 */
}

.box9:hover {
  transform: rotate(45deg); /* 围绕右下角旋转 */
}

时间函数（Timing Functions）

时间函数控制动画的速度曲线，决定了动画如何加速或减速。

  • linear：匀速
  • ease：慢-快-慢（默认）
  • ease-in：慢开始
  • ease-out：慢结束
  • ease-in-out：慢开始和结束
  • cubic-bezier(n,n,n,n)：自定义贝塞尔曲线
/* 不同时间函数的效果 */
.box p:nth-child(1) { transition-timing-function: linear; }
.box p:nth-child(2) { transition-timing-function: ease; }
.box p:nth-child(3) { transition-timing-function: ease-in; }
.box p:nth-child(4) { transition-timing-function: ease-out; }
.box p:nth-child(5) { transition-timing-function: ease-in-out; }
.box p:nth-child(6) { transition-timing-function: cubic-bezier(0,-0.77,.99,1.66); }

Animation 关键帧动画

@keyframes 规则用于定义动画的关键帧，描述动画在不同阶段的状态。

基本语法

@keyframes animationName {
  from {
    /* 初始状态 */
  }
  to {
    /* 结束状态 */
  }
}

/* 或者使用百分比 */
@keyframes animationName {
  0% {
    /* 初始状态 */
  }
  50% {
    /* 中间状态 */
  }
  100% {
    /* 结束状态 */
  }
}

Animation 实际应用示例

  • 旋转动画

    .box1 {
  animation: rotate 3s linear 0s infinite alternate;
}

@keyframes rotate {
  from {
    transform: rotate(0deg);
  }
  to {
    transform: rotate(360deg);
  }
}

  • 左右移动

    .box2 {
  animation: moveLeftRight 3s linear 0s infinite alternate;
}

@keyframes moveLeftRight {
  from {
    transform: translateX(0);
  }
  to {
    transform: translateX(1000px);
  }
}

  • 旋转动画（使用 transform-origin 控制旋转中心点）

    .box8 {
  animation: rotateFromCorner 2s linear 0s infinite;
  transform-origin: top left; /* 指定旋转中心点为左上角 */
}

@keyframes rotateFromCorner {
  from {
    transform: rotate(0deg);
  }
  to {
    transform: rotate(360deg);
  }
}

  • 形状变换（保持最终状态）

    .box3 {
  animation: changeToCircle 1s linear 0s forwards;
}

@keyframes changeToCircle {
  from {
    border-radius: 0;
  }
  to {
    border-radius: 50%;
  }
}

  • 多颜色渐变

    .box4 {
  animation: changeColor 10s linear 0s;
}

@keyframes changeColor {
  0% { background-color: red; }
  20% { background-color: yellow; }
  40% { background-color: blue; }
  60% { background-color: green; }
  80% { background-color: purple; }
  100% { background-color: orange; }
}

动画性能优化

  • 优先使用 transform 和 opacity：这些属性不会触发重排（reflow）和重绘（repaint），性能更好

    • 合成层原理：当修改 transform 和 opacity 时，浏览器只需要在合成阶段处理这些变化，而不需要重新计算布局（Layout）或绘制（Paint）。这使得动画更加流畅，因为只有少数像素需要重新计算。
    • GPU 加速：现代浏览器会将涉及 transform 和 opacity 的元素提升到单独的合成层（composite layer），利用 GPU 进行硬件加速，从而显著提高动画性能。

  • 使用 will-change 属性：提前告知浏览器哪些属性将发生变化，以便进行优化

  • 避免过度复杂的动画：过多的动画可能影响用户体验和性能

  • 使用 requestAnimationFrame：对于 JavaScript 控制的动画，使用此 API 获得最佳性能

优秀动画库推荐

  • Animate.cssopen in new window：一个功能丰富的 CSS 动画库，提供了大量预设的动画效果，只需添加相应的类名即可使用。

  • Framer Motionopen in new window：React 生态中最受欢迎的动画库之一，提供简单易用的 API 和强大的动画功能。

  • GSAP (GreenSock Animation Platform)open in new window：功能强大的 JavaScript 动画库，支持精确控制和高性能动画，适用于复杂项目。

  • Lottieopen in new window：由 Airbnb 开发的动画库，可以将 After Effects 动画直接导入到网页中，支持 JSON 格式的动画数据。

  • Velocity.jsopen in new window：轻量级的 JavaScript 动画引擎，性能优异，API 简洁，兼容性良好。

性能对比总结

  • 高效属性

    • transform：位置、缩放、旋转等变换
    • opacity：透明度变化
    • 这些属性只触发合成（Composite）阶段，不触发布局（Layout）和绘制（Paint）

  • 低效属性

    • width, height：尺寸变化
    • left, top：位置变化（非 transform）
    • margin, padding：边距变化
    • 这些属性会触发完整的渲染流水线（Layout → Paint → Composite）

总结

  • Transition

    • 适用于简单的状态变化
    • 需要用户交互触发
    • 语法简洁，易于实现

  • Animation

    • 适用于复杂的关键帧动画
    • 可以自动播放，无需交互
    • 提供更多控制选项

  • 性能考虑

    • 优先使用 transform 和 opacity
    • 合理设置动画参数
    • 注意动画对用户体验的影响
    • 利用合成层和 GPU 加速
